# 适用范围

欢迎使用飞鸟集。飞鸟集专注于魔法上网，旨在让每位麻瓜都能感受到科技的力量。

魔法能够在以下设备中进行使用：

- [macOS](/docs/macos)
- [Windows](/docs/windows)
- [iOS](/docs/ios)
- [Android](/docs/android)


| 订阅计划 | 设备数 | 每月流量* | 价格 |
| -------- | ------ | --------- | ---- |
| 月度     | 2      | 50G       | 23   |
| 季度     | 2      | 50G       | 65   |
| 半年     | 3      | 50G       | 120  |
| 一年     | 5      | 100G      | 230  |
| 团队版   | -      | -         | 单独定制 |

\* 未使用流量不累积，每月将按照购买日重置流量。
